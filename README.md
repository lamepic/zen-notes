# Zen Notes

> A minimal in browser note editor

Zen notes allow you to create notes directly from your browser. It is minimal and easy to use.

![](header.png)

## Running Locally

1. Clone Repo

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
